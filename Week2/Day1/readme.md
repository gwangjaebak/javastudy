## 2. JSTL & EL(Expression Language)

<%= %> : 자바코드(리터럴, 연산식, 메서드 호출)
${ } : 자바코드X, 이름으로 찾는것 즉, "vo"

표현식은 name, object 쌍으로 구성되어있으며 object가 no등을 갖고 있다. 따라서 getNo()를 통해 호출해줘야 한다.

JSTL : JSP Standard "Tag" Library이다. tag에 주목하자.
자바 코드를 없애는게 목적 tag를 쓰면 계층이 있어서 가독성이 좋아진다.
el도 마찬가지

1)Core
2)fmt
3)function 로 구성된다.
